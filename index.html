Work
Happy Sunshine Food Truck App High Fidelity Prototype

A team of four and I created a High Fidelity Prototype of a mobile application for the Happy Sunshine Food Truck, created on UXPin. Project involved guerrilla research, Affinity Map, Business Model Canvas, mood boards, storyboarding, wireframes, paper prototype testing, and usability testing.

Cute Closet Advertisement Flash Application

The project was to create an advertisement for a fictional company based off of an existing company. I chose to name mine Cute Closet, after the online clothing store Modcloth.com. Project requirements were to include a button that plays an audio track, an animation on a curved path, a looping animation, and a button that linked to a website.

Chinese Zodiac Calendar Flash Application

Project for one of my courses. Students were required to create a zodiac calculator or quiz game using ActionScript and Flash. User can input their birthday to receive their Chinese zodiac animal. Each zodiac in the outer ring is a button that links to the corresponding zodiac page.

Random Quote Generator jQuery and Quote API

FreeCodeCamp's Intermediate Front End Development Projects. This was the first time I used APIs in my work. Also used jQuery, HTML, and CSS. Media queries were used to style the color of the background and button, depending on screen size. When you click the button, a random quote is generated.

Weather App jQuery and Weather API

Another one of FreeCodeCamp's Intermediate Front End Development Projects. I had to figure out how to grab weather data based on the location of the user, and I used a switch statement to change up the graphic that shows.

July Talk Promo Responsive Web Design

I created a responsive website as preliminary design research for my SVG course. The final project will be to create my own unique promotional responsive poster using SVGs.

Portfolio Website Responsive Web Design

One of FreeCodeCamp's large front-end dev projects was to create a portfolio site, using Bootstrap. For my purposes, I used Bootstrap's grid system to create a responsive website.
